# **App Name**: Katha Book

## Core Features:

- Customer Management: Add, search, and manage customer profiles with details like name, phone number, and address.
- Credit Sales Tracking: Record new credit sales for each customer, including date, quantity, saree description, and sale amount.
- Payment Tracking: Add payments from customers, recording amount, date, and mode, automatically subtracting from outstanding balance.
- Outstanding Balance & Aging: Automatically calculate and display total unpaid balance and aging of pending sales for each customer.
- PDF Statement Generation: Generate professional PDF statements for customers with shop branding, transaction details, and overdue amounts.
- Dashboard & Reports: Provide a dashboard summarizing total outstanding, today's collections/sales, and customer dues, with report viewing for any date. Reports should include details about sales and collections.
- Data Storage: Store all data in Firebase Firestore, enabling real-time sync and secure backups.
- Statement tool: An LLM tool analyzes sales, payments and returns, and determines whether to flag certain line items when producing customer statements.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey trust and reliability.
- Background color: Light Lavender (#E6E6FA), a desaturated tint of deep indigo, for a soft, professional background.
- Accent color: Violet (#8F00FF), analogous to indigo but brighter and more saturated, to highlight key actions and information.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text, combining a tech-forward feel with readability. 'Space Grotesk' is a sans-serif; 'Inter' is a sans-serif.
- Use clear, professional icons to represent different functions and data points.
- Design an intuitive, mobile-first layout for easy navigation and quick access to essential information.
- Incorporate subtle animations and transitions to enhance user experience without distracting from core functionality.
